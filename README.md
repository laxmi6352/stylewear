# StyleWear

A garments business website built with React and Tailwind CSS.

## Features

- Hero landing section
- Featured categories
- New arrivals
- Newsletter signup
- Responsive design

## Getting Started

1. Clone the repository or unzip the folder
2. Install dependencies:

```bash
npm install
```

3. Run the development server:

```bash
npm start
```

## Tech Stack

- React
- Tailwind CSS
